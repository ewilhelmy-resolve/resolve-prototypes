📁 CSV/ZIP File Upload System

Upload your ticket data and access it via secure API

📋 Instructions

  • Upload CSV files containing your ticket data
  • Upload ZIP archives containing multiple CSV files
  • Maximum file size: 50MB
  • After upload, you'll receive an API key for data access
  • Use the API key to retrieve your data programmatically

🔑 API Documentation

Once you have your API key, you can access your data using the following endpoint:

Endpoint

GET http://localhost:3001/api/tickets/data

Headers

{
  "X-API-Key": "your-api-key-here",
  "Content-Type": "application/json"
}

Query Parameters

Parameter Type Description
start_date string Filter by start date (YYYY-MM-DD)
end_date string Filter by end date (YYYY-MM-DD)
status string Filter by status (open, closed, in_progress)
category string Filter by category
page number Page number (default: 1)
limit number Results per page (default: 100, max: 1000)

Example Request

curl -X GET "http://localhost:3001/api/tickets/data?status=open&page=1&limit=50" \
  -H "X-API-Key: rslv_your_api_key_here" \
  -H "Content-Type: application/json"
