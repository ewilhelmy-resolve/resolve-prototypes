# Client

React 18 + Vite + TypeScript frontend with shadcn/ui.

## Commands

```bash
pnpm --filter rita-client dev              # Dev server (port 5173)
pnpm --filter rita-client test:unit        # Run tests (watch)
pnpm --filter rita-client test:unit:run    # Run tests (once)
pnpm --filter rita-client type-check       # TypeScript check (tsgo)
pnpm --filter rita-client storybook        # Storybook (port 6006)
```

**Do NOT run `check:fix` without asking** — it reformats ALL files, not just changed ones.

## State Management

- **Server state**: TanStack Query v5 — all API data
- **Client state**: Zustand stores — UI state, local preferences
- **Forms**: React Hook Form + Zod validation

### Hook Pattern
```typescript
// Query key factory (exported for cache invalidation)
export const fooKeys = {
  all: ["foo"] as const,
  lists: () => [...fooKeys.all, "list"] as const,
  detail: (id: string) => [...fooKeys.all, id] as const,
};

// Query hook
export function useFoo(params) {
  return useQuery({ queryKey: fooKeys.list(params), queryFn: () => fooApi.list(params) });
}

// Mutation hook
export function useCreateFoo() {
  const queryClient = useQueryClient();
  return useMutation({
    mutationFn: (data) => fooApi.create(data),
    onSuccess: () => queryClient.invalidateQueries({ queryKey: fooKeys.lists() }),
  });
}
```

### Store Pattern
```typescript
export const useFooStore = create<FooState>()(
  devtools((set) => ({
    value: null,
    setValue: (v) => set({ value: v }),
  }))
);
```

## UI Components

- **`components/ui/`** — shadcn/ui primitives (button, dialog, sheet, tabs, etc.). Generated by shadcn CLI.
- **`components/custom/`** — Rita-specific components (spinner, confetti, feedback-banner, multi-select, status-alert)
- **`components/chat/`**, **`components/tickets/`**, etc. — feature components

## i18n

- `useTranslation` hook with namespace: `const { t } = useTranslation("chat")`
- Locale files: `src/i18n/locales/en/*.json` and `src/i18n/locales/es-MX/*.json`
- Interpolation: `t("key", { count: 5 })`

## Routing

- `createBrowserRouter` in `src/router.tsx`
- Protected routes wrapped in `<ProtectedRoute>` or `<RoleProtectedRoute>`
- Iframe routes at `/iframe/chat` (host-delegated auth via Valkey sessionKey)

## Testing

- **Framework**: vitest + @testing-library/react
- **Pattern**: mock all hooks/stores with `vi.mock`, render component, assert
- **Location**: `.test.tsx` next to source file

## Key Directories

| Directory | Contains |
|-----------|----------|
| `src/pages/` | Page components (one per route) |
| `src/components/ui/` | shadcn/ui primitives |
| `src/components/custom/` | Rita-specific reusable components |
| `src/hooks/api/` | TanStack Query hooks per resource |
| `src/stores/` | Zustand stores |
| `src/services/api.ts` | API client with `apiRequest<T>()` |
| `src/i18n/` | Translations |
