🔐 Quick Token Setup Guide

Follow these simple steps to generate your API token:

  1. 👆
    Click "Create API token"

    Look for the blue button on the page below

  2. ✏️
    Name your token

    Use "Resolve-Integration" or something memorable

  3. 📅
    Set expiry date

    Choose 1 year (maximum allowed for security)

  4. ✅
    Click "Create"

    Your token will appear only once!

  5. 📋
    Copy the token

    Click the copy button or select all and copy

  6. 📍
    Close this window

    Return here and paste your token

💭 Important: Save your token somewhere safe - Jira won't show it again!
